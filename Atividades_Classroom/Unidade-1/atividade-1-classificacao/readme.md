# Atividade 1 - Sistema de Classificação Acadêmica

## Descrição

Este projeto foi desenvolvido como parte da disciplina de **Programação Web I**.
O sistema recebe a nota de um aluno e informa sua situação final, além de exibir uma sequência de valores de 0 até a nota informada.

---

##  Funcionalidades

* Receber a nota do aluno via formulário
* Classificar o aluno em:

  * **Aprovado** (nota ≥ 7)
  * **Recuperação** (nota ≥ 5 e < 7)
  * **Reprovado** (nota < 5)
* Exibir sequência de números de 0 até a nota informada
* Validação de entrada (nota entre 0 e 10)

---

## Tecnologias utilizadas

* PHP (procedural)
* HTML5
* CSS básico

---

##  Como executar o projeto

1. Abra o terminal na pasta do projeto
2. Execute o servidor PHP:

```
php -S localhost:8000
```

3. Acesse no navegador:

```
http://localhost:8000
```
